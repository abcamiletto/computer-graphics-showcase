# Showcase CG 24

This repo contains the code for the showcase of the 2024 Computer Graphics Rendering Competition at Saarland University.
It is built with 11ty and deployed with Vercel.

## Development

The only requirement for development is 11ty. Check on the official website for installation instructions.

To start the development server, run `npx eleventy --serve`.

To deploy the website, push to the `main` branch. The website is automatically deployed with Vercel.
